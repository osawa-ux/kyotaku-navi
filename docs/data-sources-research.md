## 1. 主データソース（全国一括取得に最適）

### 1-1. 厚生労働省 介護サービス情報公表システム オープンデータ ★最有力★

**URL**
- 公開ページ: https://www.mhlw.go.jp/stf/kaigo-kouhyou_opendata.html
- 居宅介護支援CSVの直接URL（最新: 2025年12月末時点）:
  `https://www.mhlw.go.jp/content/12300000/jigyosho_430.csv`
- ファイル命名規則: `jigyosho_{サービスコード}.csv`（居宅介護支援 = 430）

**取得可能な項目（推奨データセット準拠）**

以下は「推奨データセット 介護サービス事業所一覧」の標準項目（デジタル庁基準）に準じた項目群。
保有していないデータは空白。

| カテゴリ | 主な項目 |
|----------|----------|
| 基本識別 | 都道府県コード、市区町村コード、介護保険事業所番号（10桁）、法人番号（13桁） |
| 名称・所在地 | 事業所名称、法人名、郵便番号、住所（都道府県・市区町村・番地）、電話番号、FAX |
| 所在地（位置情報） | 緯度、経度（北海道データで確認。全国データでの有無は要CSV実物確認） |
| 事業所属性 | 事業開始年月日、事業所区分、指定年月日、廃止年月日 |
| サービス提供体制 | 利用可能曜日（一部サービス）、高齢者・障害者同時利用可否 |
| 従業者情報 | ケアマネジャー数（居宅介護支援特有）は別途詳細CSVに含まれる可能性あり |
| 営業情報 | 営業時間、休業日 |

- 法人番号は2016年度（平成28年度）以降の基本情報報告に追加済み → 国税庁法人番号との紐づけ可能
- 事業所番号（10桁）がユニークキー

**データ形式**
- CSV（UTF-8）
- 旧データはZIP圧縮、最新データは非圧縮CSV

**全国件数規模**
- 居宅介護支援事業所: **約36,895件**（2025年9月時点の販売データより推計）
- ファイルサイズ: 約14.8MB（最大規模クラスのサービス）

**更新頻度**
- 年2回: 毎年6月末時点、12月末時点（約半年ラグあり）

**一括取得の可否**
- 全国1ファイルで一括ダウンロード可能（都道府県分割不要）
- 他サービスのCSVも同一ページに並列配置

**ライセンス・利用規約**
- CC BY 4.0（クリエイティブ・コモンズ 表示 4.0）
- 出典明記で営利・非営利問わず二次利用可能
- 根拠法: 官民データ活用推進基本法

**機械取得難易度: 簡単**
- 単純なHTTP GETで直接CSVファイルをダウンロードするだけ
- 認証不要
- ロボット対策なし

**取得手順**
```python
import requests

url = "https://www.mhlw.go.jp/content/12300000/jigyosho_430.csv"
response = requests.get(url)
with open("kyotaku_kaigo_shien.csv", "wb") as f:
    f.write(response.content)
```

## 2. 補助データソース（詳細情報・補完用）

### 2-1. 大阪府 介護保険事業所台帳情報

**URL**: https://www.pref.osaka.lg.jp/o090100/jigyoshido/kaigo/data.html

**特徴**
- 各指定権者（府・市町村）が厚生労働省仕様に基づき作成したデータ
- 居宅介護支援を含む居宅サービス等がCSV提供
- 項目説明のExcelファイルも同梱（項目一覧確認に有用）
- ファイルサイズ: 居宅サービス等1 (17,216KB) + 居宅サービス等2 (18,516KB)

**更新頻度**: 月初確定情報を当月25日を目途に更新（月次更新）

**機械取得難易度: 簡単**（直接CSVダウンロードリンク）

**用途**: 厚労省オープンデータより更新頻度が高い。大阪府のデータは補完用途で有用。項目定義書を先にダウンロードしてカラム構造を確認することを推奨。

---

### 2-2. 北海道 介護サービス事業所データ（位置情報付き）

**URL**: https://www.harp.lg.jp/opendata/dataset/1739.html

**特徴**
- **緯度経度付き**が明記されている数少ない公式ソース
- CSV形式、CC BY、3.21MB
- GIS・地図活用用途に特化
- 更新: 2026年2月（比較的新しい）
- ダウンロード数12,918件と利用実績あり

**用途**: 緯度経度データが必要な場合の参照モデル。厚労省の全国CSVに緯度経度が含まれない場合、ジオコーディングの代替として参照可能。

---

### 2-3. data.go.jp（政府オープンデータカタログ）

**URL**: https://www.data.go.jp/data/dataset/mhlw_20210301_0014

**特徴**
- 厚労省オープンデータの公式カタログ登録版
- 令和2年12月末版などの過去データも保存・参照可能
- 厚労省直リンクとは別に永続的なカタログURLとして管理
- 最新データは厚労省直URLの方が確実

**用途**: 過去データとの比較、データ系統の確認用。

## 3. 後回しデータソース（優先度低）

### 3-1. 介護サービス情報公表システム（サイトスクレイピング）

**URL**: https://www.kaigokensaku.mhlw.go.jp/

**詳細情報の取得はここが唯一のソース**だが、CSVに含まれない詳細項目（ケアマネ人数詳細、特色、特定事業者加算状況等）を取得する場合に検討。

**URLパターン**
```
https://www.kaigokensaku.mhlw.go.jp/{都道府県コード2桁}/index.php
  ?action_kouhyou_pref_search_list_list=true
  &ServiceCd=430
  &PrefCd={都道府県コード}
```

詳細ページ:
```
https://www.kaigokensaku.mhlw.go.jp/{PrefCd}/index.php
  ?action_kouhyou_detail_{バージョン}_kihon=true
  &JigyosyoCd={事業所番号}-00
  &ServiceCd=430
```

**スクレイピング難易度: 難しい**
- SPA（Single Page Application）のためSelenium必須（requests単体では不可）
- ヘッドレスモードで動作しない場合がある（ユーザーエージェント偽装が必要）
- 50件表示設定 → ページネーションループで全件取得が必要
- 都道府県47件 × 平均800件/都道府県 ≒ 37,600リクエスト規模

**公式APIなし**（非公開）

## 4. 実装推奨フロー

### Step 1: 全国ベースデータ取得（厚労省CSV）

```
GET https://www.mhlw.go.jp/content/12300000/jigyosho_430.csv
→ 約36,895件の全国居宅介護支援事業所一覧を取得
→ 項目: 事業所番号、法人番号、名称、住所、電話、指定日 等
```

### Step 2: 緯度経度の補完

厚労省CSVに緯度経度が含まれない場合:
- 住所フィールドからジオコーディング（国土地理院API or Google Maps Geocoding API）
- 北海道データを参照して緯度経度カラムの存在を事前確認すること

### Step 3: 詳細情報の補完（必要な場合のみ）

ケアマネ人数・特色等が必要な場合は情報公表システムのスクレイピングを実施。
ただし工数大のため、MVP段階ではCSVの基本情報のみで運用推奨。

### Step 4: 更新管理

- 年2回（6月末・12月末）に自動でCSVを再取得して差分更新
- 事業所番号をユニークキーとして新規/廃止/変更を管理

---

## 5. 重要確認事項（実装前に必須）

1. **緯度経度の有無**: jigyosho_430.csv を実際にダウンロードして1行目のカラムを確認
2. **法人番号の有無**: 同上
3. **ケアマネ人数**: 別ファイル（詳細情報CSV）に格納されている可能性。厚労省ページで他ファイルの存在確認
4. **CC BY表示義務**: サイト上で「データ出典: 厚生労働省 介護サービス情報公表システム」の明記が必要
